# Flappy-Bird
a very simple flappy bird webpage includes
- leaderboard
- database
- flappy bird
- personal score system
- name system

Instructions:
run `$ git clone https://github.com/Alvinyuen123/Flappy-Bird.git`
then `$ npm install`
and then the others:
`$ npm run build`
`$ npm run dev` or `$ npm run start`
